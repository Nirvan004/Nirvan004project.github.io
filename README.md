IP Address Tracker
A HTML, CSS, and JavaScript-based application that tracks and displays the location of any IP address or domain using the IP Geolocation API by IPify and LeafletJS for map rendering.
This project demonstrates API integration, dynamic UI updates, and responsive front-end design.

Features:
-Gets the user's current IP address automatically on page load
-Search for any IP address or domain
-Display:
  -IP Address
  -Location (city, region)
  -Timezone
  -ISP
-Display interactive map with custom marker for the IP location
-Responsive design for mobile and desktop
-Form validation and error handling
